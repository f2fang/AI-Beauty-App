# AI Beauty App

AI-powered iOS skin analysis application built with SwiftUI, FastAPI, OpenCV, and DeepFace.

## Features

- Facial image upload
- Skin score analysis
- Skin age estimation
- Age and gender prediction
- Brightness analysis
- Redness analysis
- Texture analysis
- Pigmentation analysis

## Tech Stack

### Frontend
- SwiftUI
- iOS

### Backend
- FastAPI
- OpenCV
- DeepFace
